## api reference 

### produtos 

#### GET/produtos
-**Descriçao**: obtém uma lista de produtos
-**Responde**: Array de produtos

#### POST /produtos
-**Descrição**: Cria um novo produto
-**Body**:
```
{
    "nomeProduto": "produtoExemplo"
    "precoProduto": -.00
}
```
-**Response**:
```
{
    "message": "produto cadastrado com sucesso"
}
```
